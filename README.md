# 💰 Expense Tracker

A simple and responsive Expense Tracker built using **HTML, CSS, and JavaScript**. This project allows users to record their expenses, calculate the total spending, and store data using the browser's Local Storage.

## 🚀 Features

- Add new expenses
- Delete existing expenses
- Automatically calculates total expenses
- Data saved using Local Storage
- Clean and responsive user interface
- Beginner-friendly project

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Local Storage API

## 📂 Project Structure

```
Expense-Tracker/
│── index.html
│── style.css
│── script.js
│── README.md
```

## 📖 How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Open `index.html` in your web browser.
4. Start adding your expenses.
